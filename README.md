
# SK4ConfigKit

SK4ConfigKitは、設定画面っぽい機能をそこそこ簡単につくれるようになるフレームワークです。

SK4Toolkitを使用しています。


## 動作環境

|環境	|情報			|
|-------|---------------|
|Xcode	|7.3 (7D175)	|
|iOS	|9.0			|
|Swift	|2.2			|
|Date	|2016/4/18 -	|

## 使い方

### Cocoapodsを使ってインストール

Podfileにこんな感じで記入して下さい。

	pod 'SK4ConfigKit'

### プロジェクトをダウンロードしてインストール

GitHubからプロジェクトをダウンロードして使って下さい。
